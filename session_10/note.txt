        symbol
    reference : assign lầu đầu sẽ tạo 1 ô nhớ mới trong bộ nhớ, assign những lần sau sẽ sửa vào ô nhớ đã tạo từ lần đầu

        function

        undefined

1.Function
    - JS function là 1 Function object

        var b = a;
        console.log(typeof b); //function
    - khai báo function
        + declaction function
            function foo() {...}
        + expression function
            var foo = function() {...}
        + self-involing function
            (function() {...})()
    - 'this' trong function = context mà nó đc gọi
        var a = {
            name: "Duy",
            age: 21,
            showInfor: function() {
                console.log(this); // 'this' thay thế  cho a
            }
        }
        var a = Person("Duy", 21);
